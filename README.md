# Digital Wedding Invitation — Template 1

This repository contains a simple responsive wedding invitation template.

Preview
- Open `index.html` in a browser, or serve the folder and visit `http://localhost:8000`.

Quick preview (PowerShell):

```powershell
python -m http.server 8000
```

Files
- `index.html` — main entry
- `admin.html` — admin view
- `assets/` — CSS, images and JS
